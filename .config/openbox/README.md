To make openbox show all your installed applications,you can install <b>[MenuMaker](http://menumaker.sourceforge.net/)</b> and run the following command:

```
mmaker -vf OpenBox3
```

optional: obmenu-generator

| Shortcut Key | App |

| mod4 + b  | Open Brave Browser  |
